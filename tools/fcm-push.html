FCM Web Push test

Dán Firebase web config + VAPID key → xin permission → lấy FCM token → gửi test message từ Console để xem có nhận noti không.

Lưu ý: Tool này có dùng Firebase CDN + FCM (đúng mục đích test). Cần HTTPS hoặc localhost. Config chỉ lưu localStorage trên máy bạn.

  1. Firebase Console → Project settings → Your apps → Web config
  2. Cloud Messaging → Web Push certificates → Key pair (VAPID Key pair)
  3. Bấm Start / Get token, cho phép Notification
  4. Console → Messaging → tạo campaign / Send test message → dán token
  5. Tab đang mở: foreground payload hiện bên dưới. Tab ẩn / nền: SW hiện system notification.
